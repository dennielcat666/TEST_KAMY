# Тестовое задание для Ассоциации «КАМИ»

Верстка адаптивная, кроссбраузерная, проверена на W3C и через PixelPerfect.

## Стек

1. Gulp
2. PUG
3. SASS
4. jQuery
5. Babel

## Сборка

Дев сборка
```bash
gulp dev
```
Watch режим
```bash
gulp watch
```
Продакшн сборка
```bash
gulp dist
```
